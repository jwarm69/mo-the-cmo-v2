# Bite Club Paid Social Media Campaign Setup Guide

## Platform-Specific Campaign Strategy

### TikTok Ads Manager Setup

#### Campaign Objective Hierarchy
1. **Awareness Campaigns** (20% of budget)
   - **Objective**: Reach
   - **Goal**: Brand awareness and consideration
   - **Budget**: $200-400/month

2. **Conversion Campaigns** (60% of budget) 
   - **Objective**: App Install
   - **Goal**: Drive app downloads
   - **Budget**: $600-1200/month

3. **Engagement Campaigns** (20% of budget)
   - **Objective**: Video Views
   - **Goal**: Content amplification and retargeting pool building
   - **Budget**: $200-400/month

#### TikTok Targeting Parameters

**Core Demographics**:
```
Age: 18-24
Gender: All genders
Location: Gainesville, FL (20-mile radius)
Language: English
```

**Advanced Targeting**:
```
Education:
- College students
- Currently enrolled in university
- University of Florida (when available)

Interests:
- Food & dining
- Fast food
- Food delivery
- Cooking
- College life
- Budget/savings apps
- Student lifestyle

Behavior:
- Recently moved (new students)
- Mobile app installers
- Food delivery users
- Price-conscious consumers
```

**Custom Audiences**:
```
Website Visitors:
- All website visitors (last 30 days)
- App download page visitors
- Blog/content page visitors

Engagement Audiences:
- Video viewers (25%, 50%, 75%, 100%)
- Profile visitors
- Content interactors

Lookalike Audiences:
- Current app users (1%, 2%, 5% lookalikes)
- Website converters
- High-value customers (multiple orders)
```

#### TikTok Creative Specifications
```
Video Format:
- Aspect Ratio: 9:16 (vertical)
- Resolution: 720×1280 or 1080×1920
- Duration: 15-60 seconds (30s optimal for ads)
- File Size: Max 500MB
- Format: MP4, MOV, MPEG, 3GP, AVI

Audio:
- Use trending sounds when relevant
- Clear voiceover or text overlay
- Music volume balanced with speech

Text Elements:
- Headline: 12-40 characters
- Description: Up to 80 characters
- CTA: "Download" or "Learn More"
```

**Visual Structure**:
```

Scene 2 (3-8s): Side-by-side comparison

Bite Club receipt: $16.00
Savings: $8.67

Scene 3 (8-12s): Monthly projection
"$8 per order × 25 orders = $200/month saved"
```

## Budget Allocation & Bidding Strategy

### Monthly Budget Distribution
```
Total Monthly Ad Spend: $1,000-2,000

Platform Split:
- TikTok Ads: 40% ($400-800)
- Instagram/Facebook: 45% ($450-900)
- YouTube Shorts: 15% ($150-300)

Campaign Type Split:
- App Install: 60%
- Traffic/Awareness: 25% 
- Retargeting: 15%
```

### Bidding Strategy by Objective

#### App Install Campaigns
```
TikTok:
- Bid Strategy: Cost Cap
- Target: $3-5 per app install
- Budget: $50-100/day
- Optimization: App Install

Instagram/Facebook:
- Bid Strategy: Lowest Cost with Cost Cap
- Target: $2-4 per app install  
- Budget: $40-80/day
- Optimization: App Install
```

#### Traffic Campaigns
```
Cost Per Click Targets:
- TikTok: $0.50-1.00 CPC
- Instagram: $0.30-0.80 CPC
- Facebook: $0.25-0.60 CPC

Landing Page Optimization:
- Fast load times (<3 seconds)
- Mobile-optimized experience
- Clear value proposition
- Prominent app download buttons
```

### Campaign Testing Framework

#### Weekly Testing Schedule
```
Monday: Launch new creative tests
Tuesday: Analyze Monday performance, adjust budgets
Wednesday: Mid-week performance review
Thursday: Optimize top performers, pause poor performers
Friday: Weekend creative prep and audience expansion
```

#### A/B Testing Variables
```
Creative Elements:
- Hook variations (first 3 seconds)
- Visual style (students vs food focus)
- CTA placement and wording
- Music/audio choices

Targeting Options:
- Broad vs detailed targeting
- Age range variations (18-22 vs 18-24)
- Interest combinations
- Geographic radius (15 vs 20 miles)

Bidding & Budget:
- Manual vs automatic bidding
- Daily budget amounts
- Campaign optimization events
- Delivery optimization
```

## Advanced Campaign Optimizations

### Audience Segmentation Strategy

#### High-Intent Audiences (Higher budgets, lower CPC targets)
```
Characteristics:
- Visited app download page
- Engaged with multiple videos
- Completed video views (75%+)
- Similar to current customers

Budget Allocation: 40% of total spend
Target CPC: 20% below account average
```

#### Discovery Audiences (Broader reach, higher CPC tolerance)
```
Characteristics:
- Broad interest targeting
- Lookalike audiences (5%+)
- New to brand/competitor audiences
- Campus-wide geographic targeting

Budget Allocation: 35% of total spend
Target CPC: Account average or 10% above
```

#### Retargeting Audiences (Highest conversion rates)
```
Characteristics:
- Website visitors (last 30 days)
- Video viewers who didn't convert
- App downloaders who didn't register
- Previous customers (re-engagement)

Budget Allocation: 25% of total spend
Target CPC: 30% below account average
```
